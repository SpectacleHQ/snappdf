# Privacy Policy

SnapPDF is an open-source desktop application for exporting PDF pages to image files.

## Local Processing

SnapPDF processes PDF files locally on the user's device. The application does not require an account, online service, cloud upload, or remote server to convert PDF pages to images.

## Data Collection

SnapPDF does not collect, store, sell, transmit, or share personal information.

SnapPDF does not upload PDF files, exported images, document contents, filenames, file paths, usage data, analytics, or telemetry to any remote server.

## Network Access

The current application logic does not require network access for PDF conversion.

If future versions add features that require network access, this policy and the README will be updated before those features are released.

## Files Accessed by the Application

SnapPDF only accesses files selected by the user, such as:

- PDF files selected for conversion
- Output directories selected by the user
- Image files generated by the conversion process

## Third-party Services

SnapPDF does not use third-party analytics, advertising SDKs, tracking services, or cloud document processing services.
